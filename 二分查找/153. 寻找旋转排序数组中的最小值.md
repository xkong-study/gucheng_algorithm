这种线性模式的题目也是二分查找的，mid一直和right进行比较。

```code
var findMin = function(nums) {
let left = 0;
let right = nums.length-1;
while(left<right){
    mid = left+Math.floor((right-left)/2);
    if(nums[mid]<nums[right]){
    right = mid;
    }
    else {
    left = mid+1;
    }
}
return nums[left];
};
```
