***约瑟夫环的递推公式意思是：**

***“上一个安全位置 + k，绕圈取模”***

也就是每轮报数跳过 k 个人，然后从剩下的人中继续。      

**1.递推方法：你只记住这一句口诀就能写出递推了**：      

当前位置 = (上一次的位置 + 步长 k) % 当前人数        


**2.数学方法；x是总人数 y是当前的位置**

```
function josephus(n, k) {
  let res = 0; // f(1, k) = 0
  for (let i = 2; i <= n; i++) {
    res = (res + k) % i;
  }
  return res;
}


它相当于：

i = 2: f(2) = (f(1) + k) % 2

i = 3: f(3) = (f(2) + k) % 3

...

i = n: f(n) = (f(n-1) + k) % n
```


**链表**

```
class Node {
  constructor(val) {
    this.val = val;
    this.next = null;
  }
}

function josephusLinkedList(n, k) {
  let head = new Node(0);
  let curr = head;

  // 构造循环链表
  for (let i = 1; i < n; i++) {
    curr.next = new Node(i);
    curr = curr.next;
  }
  curr.next = head; // 尾巴连到头部，形成环

  // 模拟删除过程
  let prev = curr; // 最后一位
  curr = head;
  while (curr !== curr.next) {
    // 往后走 k - 1 步，删第 k 个
    for (let i = 1; i < k; i++) {
      prev = curr;
      curr = curr.next;
    }
    // 删除 curr（第 k 个）
    prev.next = curr.next;
    curr = curr.next;
  }

  return curr.val;
}
```
