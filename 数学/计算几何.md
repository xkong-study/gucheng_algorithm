🔹 核心判断逻辑：     

判断“是否左转”就是：看新点相对于当前路径的延长线，是在左边还是右边。

🔸 操作步骤：          

当前路径从 A ➝ B 形成向量 AB       

下一个候选点为 C，形成 BC       

用向量叉积判断方向：         

cross(AB,BC)=(B−A)×(C−B)     

结果 > 0 → 左转 ✅ → 保留路径       

结果 < 0 → 右转 ❌ → 弹出旧点      

结果 = 0 → 共线（根据需求处理）

###  工具函数常备：
```js
// 判断是否三点共线
function isColinear(x1, y1, x2, y2, x3, y3) {
  return (y2 - y1) * (x3 - x2) === (y3 - y2) * (x2 - x1);
}

// 判断两向量是否垂直（点积为 0）
function isPerpendicular(ax, ay, bx, by) {
  return ax * bx + ay * by === 0;
}

// 构造两点的中点坐标
function midpoint(x1, y1, x2, y2) {
  return [(x1 + x2) / 2, (y1 + y2) / 2];
}

// 计算最大公约数 gcd
function gcd(a, b) {
  return b === 0 ? a : gcd(b, a % b);
}
```

###  涉及到的算法与知识点（矩形/多边形专题）

#### ▫️ 1. 矩形重叠判断
- 两个轴对齐矩形有重叠 ⇔ 1D 投影在 x 和 y 方向都有交集

<img width="661" alt="Screenshot 2025-04-13 at 21 30 38" src="https://github.com/user-attachments/assets/3c369403-14bc-4a85-b975-5a3373288886" />


#### 🔲 2. 面积容斥原理
- 总面积 = 面积 A + 面积 B − 重叠面积
- 重叠部分 = x 区间交集 × y 区间交集

#### 🔺 3. 正方形判定
- 四边长度相等：用点之间距离平方判断
- 两条对角线也需相等
- 正交判断：使用向量点积 = 0 判直角

#### 🔹 4. 枚举矩形 + 哈希优化
- 利用坐标哈希加速找候选点（如最小面积矩形）
- 可构造边长、角度、中心点作为判重键

#### 🔁 5. 中点+向量旋转判矩形 II
- 向量转角度、旋转构造四边形
- 中点哈希 + 长度平方计算面积

#### 🧭 6. 多边形是否凸（凸包性质）
- 每三个连续点构成两个向量
- 判断所有相邻向量叉积方向是否一致 ⇒ 若一致则为凸

### 相关题目与所用性质：

---

### 涉及到的算法与知识点（凸包专题）

#### 🧱 1. 凸包的定义
- 凸包是包住所有点的最小凸多边形
- 常用于边界分析、围栏规划、最远点对、凸性判定等

#### 📐 2. Graham 扫描算法
- 按极角排序（以最低点为基准）
- 逐步加入点，用栈维护当前凸壳
- 判断是否「右转」：若是则出栈，直到「左转」为止

#### 📈 3. Andrew 单调链算法（推荐）
- 将点按 x 升序排序
- 构造下凸包（左到右）+ 上凸包（右到左）
- 每步用向量叉积判断是否左转

#### ⛳ 4. 凸包判定方向（叉积）
- 给定三点 A, B, C，若叉积 > 0 ⇒ 左转，< 0 ⇒ 右转
- 使用方向一致性判断凸性 / 剔除不合法点

#### 🧭 5. 点是否在多边形内部（射线法 / 叉积法）
- 应用于判断某点是否在凸包中（如 LCP 15）
- 方法一：向右发射射线，看穿过边数是否为奇数
- 方法二：逐边叉积方向是否一致

### 工具函数示例
```js
// 向量叉积
function cross(ax, ay, bx, by) {
  return ax * by - ay * bx;
}

// 凸包构造（Monotone Chain）
function convexHull(points) {
  points.sort((a, b) => a[0] - b[0] || a[1] - b[1]);
  const cross = (o, a, b) => (a[0] - o[0]) * (b[1] - o[1]) - (a[1] - o[1]) * (b[0] - o[0]);
  const lower = [];
  for (const p of points) {
    while (lower.length >= 2 && cross(lower[lower.length - 2], lower[lower.length - 1], p) <= 0) lower.pop();
    lower.push(p);
  }
  const upper = [];
  for (let i = points.length - 1; i >= 0; i--) {
    const p = points[i];
    while (upper.length >= 2 && cross(upper[upper.length - 2], upper[upper.length - 1], p) <= 0) upper.pop();
    upper.push(p);
  }
  upper.pop(); lower.pop();
  return lower.concat(upper);
}
```
