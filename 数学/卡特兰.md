### ✅ 二、Dyck 路径定义

**Dyck 路径**是从 \((0, 0)\) 到 \((2n, 0)\)，由 n 步 \(+1\) 和 n 步 \(-1\) 构成的路径，要求：

- 任何前缀和 \(\geq 0\)，即路径不跌破 x 轴
- 总步数为 2n，终点回到原点
- 对应于栈的合法出入操作序列或括号合法匹配序列

#### 🔍 约束条件（合法 Dyck 路径）：

- 要求前缀和不能为负 ⇒ 每一步都不跌破 x 轴

#### 📐 不合法路径的数量：反射法推导

1. 任意一条“不合法路径”，在第一次跌破 x 轴的地方，把前缀镜像
2. 得到从 \((0, 0)\) 到 \((2n, -1)\) 的路径
3. 这种路径必须有：
   - n–1 次 `+1`

### ✅ 六、Catalan 数的计算代码（JS）

```js
function catalan(n) {
  const MOD = 1e9 + 7;
  let fac = Array(2 * n + 1).fill(1);

  for (let i = 1; i <= 2 * n; i++) {
    fac[i] = fac[i - 1] * i % MOD;
  }

  const modInv = (a, mod) => {
    let res = 1, b = mod - 2;
    while (b) {
      if (b & 1) res = res * a % mod;
      a = a * a % mod;
      b >>= 1;
    }
    return res;
  };

  return fac[2 * n] * modInv(fac[n], MOD) % MOD * modInv(fac[n + 1], MOD) % MOD;
}
```

# 📚 卡特兰数（Catalan Number）递推公式与组合解释

## ✅ 定义

卡特兰数 Cₓ 表示多种递归结构的总数，比如：

- n 对括号的合法组合数量  
- n 个节点的二叉树结构数  
- 从 (0,0) 到 (2n,0) 的合法 Dyck 路径数  
- 栈操作合法出栈序列数量  
- 非交发配对方案数（如 n 条弦不相交连接）

## ✅ 动态规划 JS 实现

```js
function catalan(n) {
  const dp = Array(n + 1).fill(0);
  dp[0] = 1;
  for (let i = 1; i <= n; i++) {
    for (let j = 0; j < i; j++) {
      dp[i] += dp[j] * dp[i - 1 - j];
    }
  }
  return dp[n];
}
```
